# **App Name**: GradeWise

## Core Features:

- Subject Input: Allow students to manually input subject details including name, credits, and marks or status.
- GPA Calculation: Automatically calculate GPA based on marks, credits, and a 10-point grading system.
- CGPA Calculation: Calculate CGPA using data from all semesters to reflect cumulative academic performance.
- Automated insights: Provide students with clear, actionable insights on which classes they should focus more on to increase their overall score.
- AI-Powered Status Prediction: Use AI to predict potential subject status (PASS, RA, etc.) based on historical performance. This can be used as a tool for students to identify subjects they might struggle with early on.
- Degree Classification: Determine and display the final degree classification (e.g., First Class - Exemplary) based on the calculated CGPA.
- Data Storage: Save each semester, grade and result in a SQL database for future calculations and auditing

## Style Guidelines:

- Primary color: Deep sky blue (#3498DB) to evoke trust and stability.
- Background color: Light gray (#ECF0F1), nearly white, to give a clean and modern feel.
- Accent color: Emerald green (#2ECC71) to highlight positive outcomes and important actions.
- Body and headline font: 'Inter', a grotesque sans-serif font known for a neutral and modern aesthetic, for clarity.
- Use minimalistic, line-based icons to represent subjects and status indicators for clarity.
- Clean and structured layout to clearly present subject data, GPA, CGPA, and degree classification.
- Subtle animations for loading and updating calculations.
